# Emergence and Limits of Successor Attractors

Reference implementation and draft manuscript for:

**Learning Discrete Successor Transitions in Continuous Attractor Networks:
Emergence, Limits, and Topological Constraints**

This repository accompanies a preprint intended for arXiv submission.
It explores learning successor-like displacement dynamics in continuous
attractor networks, including failure modes and topological effects.

Status: preprint draft  
Author: Daniel Brownell
arxiv url: http://arxiv.org/abs/2601.15336
